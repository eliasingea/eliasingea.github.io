Latest Projects

LebGeo

LebGeo was a project that I had started working on as a senior in high school. The idea was simple, create a simple way for people to get turn-by-turn navigation in areas that lacked the proper resources such as GPS and proper road and geospatial data. By using opensource data I was able to create a road network of Beirut, Lebanon that would continuously be improved and create a way for people to get accurate directions. The goal for the project was to ensure that people had a proper way to get to safe locations in time of need.

DidTheyFeed

DidTheyFeed was a project that I had worked on that implemented the Riot Games API to get user data for the purpose of finding out whether a player has played with another and if so, how well they performed. There were endless amount of WebApps that allowed players to get statistics on other players, however what was lacking was the ability to check if you as a player had ever played with another player.
